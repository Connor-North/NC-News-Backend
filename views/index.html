NC News API Documentation

Welcome to the API documentation for NC News. Below you will find details about the available endpoints:

Endpoints

1. Get API Info

Endpoint: GET /api

Description: Serves up a JSON representation of all the available endpoints of the API.

2. Get Topics

Endpoint: GET /api/topics

Description: Serves an array of all topics.

Response Example:

{
    "topics": [{ "slug": "football", "description": "Footie!" }]
}

3. Get Topic by Slug

Endpoint: GET /api/topics/:slug

Description: Serves a single topic object matching the given slug.

Response Example:

{
  "topic": {
    "slug": "coding",
    "description": "Code is love, code is life"
  }
}

4. Get Articles

Endpoint: GET /api/articles

Description: Serves an array of all articles.

Response Example:

{
    "articles": [
        {
          "author": "icellusedkars",
          "title": "Eight pug gifs that remind me of mitch",
          "article_id": 3,
          "topic": "mitch",
          "created_at": "2020-11-03T09:12:00.000Z",
          "votes": 0,
          "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700",
          "comment_count": 2
        }
    ]
}

5. Get Users

Endpoint: GET /api/users

Description: Serves an array of all users.

Response Example:

{
    "users": [
        {
          "username": "butter_bridge",
          "name": "jonny",
          "avatar_url": "https://www.healthytherapies.com/wp-content/uploads/2016/06/Lime3.jpg"
        }
    ]
}

6. Get Article by ID

Endpoint: GET /api/articles/:article_id

Description: Serves an object of the specified article.

Response Example:

{
    "article": {
        "article_id": 1,
        "title": "Living in the shadow of a great man",
        "topic": "mitch",
        "author": "butter_bridge",
        "body": "I find this existence challenging",
        "created_at": "2020-07-09T20:11:00.000Z",
        "votes": 100,
        "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"
    }
}

7. Get Comments for Article

Endpoint: GET /api/articles/:article_id/comments

Description: Serves an array of comments for the specified article.

Response Example:

{
    "comments": [
        {
          "comment_id": 5,
          "votes": 0,
          "created_at": "2020-11-03T21:00:00.000Z",
          "author": "icellusedkars",
          "body": "I hate streaming noses",
          "article_id": 1
        }
    ]
}

8. Add Comment to Article

Endpoint: POST /api/articles/:article_id/comments

Description: Adds a new comment to the specified article.

Request Example:

{
    "username": "butter_bridge",
    "body": "Great article!"
}

Response Example:

{
    "comment": {
        "comment_id": 20,
        "votes": 0,
        "created_at": "2025-06-05T12:34:56.000Z",
        "author": "butter_bridge",
        "body": "Great article!",
        "article_id": 1
    }
}

9. Update Article Votes

Endpoint: PATCH /api/articles/:article_id

Description: Updates an article's vote count by the given increment.

Request Example:

{
    "inc_votes": 1
}

Response Example:

{
    "article": {
        "article_id": 1,
        "title": "Living in the shadow of a great man",
        "topic": "mitch",
        "author": "butter_bridge",
        "body": "I find this existence challenging",
        "created_at": "2020-07-09T20:11:00.000Z",
        "votes": 101,
        "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"
    }
}
